# ec-eGov
전자정부 프레임워크 커스트마이징.
egovframework 3.9 버전 전체 공통 콤포넌트에서 사용하지 않는 불필요한 콤포넌트 제거 및 의존성 제거.
개선 및 기능추가 커스트마이징.

# 환경설정
  * 로컬DB는 script 디렉토리에 있는 sql 사용.
  * 로컬DB 패스워드는 EgovEnvCryptoUserTest.java 로 암호화된 db패스워드값을 globals.properties에 사용.
