# Crontab 참고

## 필드
    | 필드 이름 | 허용 값 | 
    | --- | --- |
    | 초(seconds) |  0 ~ 59 |
    | 분(minutes) |  0 ~ 59 |
    | 시(hours) |  0 ~ 23 |
    | 날(day-of-month) | 1 ~ 31 |
    | 달(month) | 1 ~ 12 or JAN ~ DEC |
    | 요일(day-of-week) | 1 ~ 7 or SUN-SAT |
    | 년도 | 빈값, 1970 ~ 2099 |
